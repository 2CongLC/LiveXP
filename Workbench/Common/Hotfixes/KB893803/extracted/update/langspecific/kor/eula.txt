MICROSOFT 소프트웨어 최종 사용자 사용권 계약서 추록 
(“사용권 계약서 추록”)

중요한 내용이므로 자세히 읽으시기 바랍니다. 본 사용권 
계약서 추록이 동봉된 Microsoft 운영 체제 구성 요소("OS 
구성 요소")는 모든 “온라인” 또는 전자 문서를 포함하여 
아래 설명된 해당 Microsoft 운영 체제 제품의 사용권 
계약서(각각 “최종 사용자 사용권 계약서” 또는 “사용권 
계약서”)와 본 사용권 계약서 추록의 적용을 받습니다. 
귀하는 OS 구성 요소를 설치, 복사, 다운로드, 액세스, 
또는 사용함과 동시에 해당 운영 체제 제품 사용권 
계약과 본 사용권 계약서 추록의 내용에 동의하게 
됩니다. 이러한 사용권 계약서의 내용에 동의하지 않을
경우에는 OS 구성 요소를 설치, 복사, 다운로드, 
액세스, 또는 사용하지 마십시오.

참고: 적법하게 사용권을 취득하였으며 각각 OS 구성
요소를 제공받은 MICROSOFT WINDOWS 운영 체제 제품
(MICROSOFT WINDOWS XP PROFESSIONAL, 
MICROSOFT WINDOWS 2000 PROFESSIONAL, 
MICROSOFT WINDOWS 2000 SERVER, MICROSOFT 
WINDOWS 2000 ADVANCED SERVER) (각각 “OS 제품”) 
사본이 없으면 OS 구성 요소를 설치, 복사, 다운로드, 
액세스, 또는 사용할 수 없으며 본 사용권 계약서 
추록상의 어떤 권리도 없습니다.

본 사용권 계약서 추록에 사용되고 있으나 여기에서 
달리 정의되지 않은 모든 용어는 관련 OS 제품 사용권 
계약서에 정의된 뜻에 따릅니다.

일반. OS 구성 요소들은 해당되는 OS 제품의 기존 기능을 
갱신, 보충 또는 대체하기 위하여 귀하에게 제공되며, OS 
구성 요소들을 설치함과 동시에 OS 제품의 일부가 됩니다. 
귀하는 해당 OS 제품(제품 명으로 참조)에 대한 OS 제품 
사용권 계약서의 조건과 본 사용권 계약서 추록에 명시된 
조건에 따라 그리고 귀하가 이러한 모든 조건들을 준수할 
것을 전제로 하여 OS 구성 요소를 사용할 수 있는 
사용권을 부여받습니다. OS 구성 요소에 적용될 수 있는
조건 중에 일치하지 않는 부분이 있을 경우에는 본 
사용권 계약서 추록의 내용이 OS 구성 요소에 대하여 
우선적으로 적용됩니다.

추가 권리 및 제한.

* 귀하가 적법한 OS 제품의 사본 여러 개에 대한 사용권을 
부여받은 경우 귀하는 그 해당되는 OS 제품의 적법한 
사본이 실행되는 모든 컴퓨터에 운영 체제 구성 요소의 
사본 1개를 복제, 설치 및 사용할 수 있습니다. 
단 귀하는 그러한 운영 체제 구성 요소의 추가 사본을 
위에 명시된 조건에 따라 사용해야 합니다.

* 외부 매체로 OS 구성 요소를 취득한 경우가 아니라면, 
귀하는 보관 목적이나 이전에 운영 체제 구성 요소가 
설치된 동일 컴퓨터에 운영 체제 구성 요소를 다시 
설치할 목적에 한하여 운영 체제 구성 요소를 한 개 
추가로 복제할 수 있습니다. Microsoft와 그 자회사 
및/또는 공급자들은 운영 체제 구성 요소와 관련한 
모든 권리, 소유권 및 제반 이권을 보유합니다. 
명시적으로 허여되지 않은 모든 권리는 Microsoft와 
그 자회사 및 공급자가 보유합니다.

해당 OS 제품 사용권 계약서에 포함된 제한적 보증
(그러한 보증이 있는 경우)은 OS 구성 요소에도 
적용됩니다. 단 해당 OS 제품 사용권 계약서에 포함된
제한적 보증기간 내에 귀하가 OS 구성 요소의 
사용권을 취득하였어야 합니다. 본 사용권 계약서
추록은 제한적 보증 기간을 연장하지 않습니다.
